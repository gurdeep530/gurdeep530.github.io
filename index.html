    Discord Gambling Bot

    This fully fledged Discord bot was created with Node.js, JavaScript and MongoDB, and functions as a virtual gambling platform for users to gamble on in a Discord server. Users can make profiles and play games like Blackjack, Slots, or Roulette, which use an in-game currency that is tracked on a database. What makes this bot interesting is the addition of Siegeline which is a live betting system for Rainbow Six Siege. This system will connect into the ranked match in real time and create betting odds based mostly off of statistics collected while the game is being played. Bettors can place bets on the result of kills creating a feel for the second screen experience. In completing this project I had the chance to build out complex backend logic, handle live data streams, build user-based authentication and economy balancing, and perfect a command driven interface. For Siegeline, I also got to spend a great deal of time exploring API integrations and live data retention which created a lot of depth and interactivity for the bot. Overall, this was a very technically rewarding experience, and one that was also community driven.

    Firefighting Helicopter Game

    The game is a single player game developed as a college project in Java with the Codename One framework. The player controls a helicopter that is putting out fires around a simulated city in the game. Fires are displayed as pink circles on top of buildings. The player needs to fly to a nearby river in order to get water, and then fly to the location of the fire to drop the water on it. The goal of the game is to put out all the fires before running out of fuel. If the player runs out of fuel the game is over. While there are many aspects to this game, one of the unique aspects of the game is the inclusion of an AI controlled helicopter. When a player clicks on a fire the AI helicopter will automatically determine the shortest route to the river, will return to fill with water, and fly to the fire location and extinguish the fire. The AI helicopter is intended to demonstrate pathfinding logic and autonomous decision-making. AI behavior enhances the interactivity of the game and adds to the overall experience.
